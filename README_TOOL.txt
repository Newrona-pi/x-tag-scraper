# Xログインツール使用方法

## 概要
このツールは、X（旧Twitter）のログインセッション情報を取得し、`twitter_state.json`ファイルとして保存します。
必要なブラウザ環境はツールが自動的に準備するため、事前のインストール作業は不要です。

## 使用方法

1. `twitter_login_tool.exe`をダブルクリックして実行します
2. **初回のみ**、ブラウザのダウンロードとセットアップが自動的に行われます（数分かかります）
3. 準備が完了するとブラウザが起動し、Xのログイン画面が表示されます
4. 通常通りXにログインしてください
5. ログインが完了すると、自動的にセッション情報が保存され、ブラウザが閉じます
6. 同じフォルダ内に`twitter_state.json`ファイルが作成されます

## ファイル構成

- `twitter_login_tool.exe`: 実行ファイル
- `bin/`: ブラウザが格納されるフォルダ（自動作成されます）
- `twitter_state.json`: 保存されたセッション情報（自動作成されます）

## 注意事項

- インターネット接続が必要です
- セッション情報は機密情報です。他人と共有しないでください
- ツールを削除する場合は、`twitter_login_tool.exe`と`bin`フォルダを削除してください

## トラブルシューティング

### ダウンロードが進まない場合
- インターネット接続を確認してください
- セキュリティソフトが通信をブロックしていないか確認してください

### エラーが発生する場合
- `bin`フォルダを一度削除してから、再度実行してみてください
